Detailed Description

Classe responsável pela persistência de dados em suporte físico (ficheiros). Permite a serialização e desserialização das listas de alojamentos, clientes e reservas.

Carrega os dados de um ficheiro binário e repovoa as listas em memória. Este processo limpa os dados atuais antes de inserir os dados carregados.

Parameters
nomeficheiroCaminho ou nome do ficheiro a ler.
Returns
true se os dados forem carregados; false se o ficheiro não for encontrado.
Exceptions
ArgumentExceptionLançada se o nome do ficheiro for nulo ou vazio.
CarregarDadosExceptionLançada em caso de erro de leitura ou desserialização.
